- yaycoinとは -

yaycoinはソロマイニング練習用のローカルコインです。
少しでも多くの人にソロ堀りのすばらしさを伝えるため制作しました。
名前はcpuminerでのブロック発見時の「yay!!!」から拝借しています。

yaycoinはDifficultyが増えれば増えるほどコインにつくボーナスが増します。
報酬は100 + 0.02441406 YAYからのスタートになり、
たくさんのPCでyay!!!するとボーナスがグングン上昇していきます。

yaycoinはohac氏制作のSha1coinがベースになっています。
Sha1coinにあるウォレットでのトリップ検索の出力は、Sha1coinオリジナルの機能として
yaycoinではカットしてあります。

ローンチは2014/03/30の12:00 JST。

  • Algorithm : SHA-1 x 27 + XOR　　yay!!!
  • Premine : 0%　　yay!!!
  • Block Reward : 100 + 直前のDifficulty100倍ボーナス　　yay!!!
  • Block time : 30 seconds　　yay!!!
  • Difficulty : DigiShield v2.0 (1block毎に変動)　　yay!!!
  • Defolt Port : 8484　　yay!!!

ソロマイニングの方法

  • その1 コンソールを開いて、setgenerate true -1を入力する
  • その2 yaycoin.conf (Windows7ならC:\Users\ユーザー名\AppData\Roaming\yaycoin内に作成)
    にgen=1を記述する(起動すると採掘を開始)
  • その3 sha1coin-cpuminerを使う

sha1coin-cpuminerの設定

例) yaycoin.confに
server=1
rpcuser=user
rpcpassword=pass
rpcallowip=127.0.0.1
rpcport=8383

と設定した場合、 sha1coin-cpuminerのフォルダ内で新規テキストを作成、中身に
minerd -a sha1coin -o 127.0.0.1:8383 -u user -p pass

をコピペ。
保存後、拡張子をtxtからbatに変更し、実行します。
sha1coin-cpuminerでは2chのトリップも検索できます。
-f yaycoin

の追加で「yaycoin」から始まるトリップをコンソール上に出力します。
デフォルトではsha1になっており、たくさんyay!!!できますが
コンソールの情報を少なくしたい方は長めのトリップを指定すると良いかもしれません。
